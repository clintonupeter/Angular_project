Setting Up a Project
Install the Angular CLI globally:

npm install -g @angular/cli
Create workspace:

ng new [PROJECT NAME]
Run the application:

cd [PROJECT NAME]
ng serve
Angular is cross-platform, fast, scalable, has incredible tooling, and is loved by millions.
